# SuitaOne

SuitaOne is a modern hotel guest experience web app that lets guests access room service, spa bookings, housekeeping, and concierge services from a single, mobile-friendly interface.

## Tech Stack

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui

## Getting Started

```sh
npm install
npm run dev
```

## Scripts

```sh
npm run dev
npm run build
npm run preview
npm run lint
npm run test
```

## Notes

- Default dev server: http://localhost:5173
